      # HTML

      # 语义化

      语义化是用合适的标签表达清晰的结构和内容。

      • 让开发者更容易读懂，便于团队开发和维护
      • 让浏览器更容易解析
      • 让爬虫更容易识别，便于SEO

      常见语义化标签有：header、nav、aside、main、footer、section、article

      # src和href区别

      • src用于替换当前元素，指向的内容会嵌入到文档中。浏览器解析到其指向的资源时，会暂停其他资源的下载和处理。
      • href用于在当前文档和引用资源之间建立联系。浏览器解析到其指向的资源时，会并行下载资源，不会停止对其他资源的下载和处理。
