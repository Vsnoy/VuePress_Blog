# 选择要合并的 commit

这里以合并前三条 commit 记录为例，有三种方式

git rebase -i HEAD~3

git rebase -i HEAD^^^

# 合并提交

保留第一行的 pick，将第二行及以后的 pick 修改为 f 或 s 保存，退出。

pick 3ca6ec3
f 1b40566
f 53f244a
1
2
3

# 合并冲突

当合并存在有冲突时，需要执行手动合并冲突。合并完成后执行如下命令，继续进行变基操作。

git rebase --continue
1

如果你想放弃这次合并的话，执行如下命令

git rebase --abort
1

# 编辑提交信息

多条 commit 合并后的 commit，附上提交信息。

git commit --amend -m <message>
