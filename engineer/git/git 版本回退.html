# git reset

回退，并舍弃提交记录。使用需要谨慎，不建议在多人协作时使用。

git reset --hard HEAD~n

git reset --hard HEAD^..

git reset --hard <commit id>
1
2
3
4
5
git push -f
1

# git revert

回退，但保留提交记录。

git revert HEAD~n

git revert HEAD^..

git revert <commit id>
1
2
3
4
5
git push
1
