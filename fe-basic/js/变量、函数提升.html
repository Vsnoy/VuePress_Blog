# 变量、函数提升

在 ES6 之前，JS 没有块级作用域的说法，只有全局和函数作用域。 按照正常的编程逻辑来说，变量或者函数，都应该先声明，再调用。 变量提升即将变量声明提升到它所在作用域的最开始的部分。

# 变量提升

var a = 1
b = 2
console.log(a + b)
var b

var a = 1
console.log(a + b)
var b = 2

TIP

JS 中创建函数有两种方式：函数声明和函数字面量。只有函数声明才存在函数提升。

# 优先级

函数提升优先于变量提升。

function a() {
  console.log(10)
}
