# var、let 及 const 区别

# var

  • 变量提升
  • 可重复声明
  • 可重复赋值

# let

  • 块级作用域
  • 暂时性死区
  • 无变量提升
  • 不可重复声明
  • 可重复赋值

# const

  • 块级作用域
  • 暂时性死区
  • 无变量提升
  • 不可重复声明
  • 不可重复赋值
